# 1_분포 함수 및 통계량

> 확률분포: 확률변수가 특정한 값을 가질 확률을 나타내는 함수
>
> - 분포함수: 확률 변수의 누적 분포를 기술하는 함수 `F(x) = 확률변수 X가 x이하의 값을 가질 확률)`
> - 확률질량함수: 이산형 데이터의 확률
> - 확률밀도함수: 연속형 데이터의 확률
